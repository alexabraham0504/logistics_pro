# Logistics Backend API

Backend API for the Bharat Logistics Platform.

## Features
- User Authentication (JWT)
- Shipment Management
- Proof of Delivery (POD)
- Transfer of Delivery (TOD)
- Blockchain Integration (Polygon Amoy)
- AI Agents (Market Intelligence, Contract Intelligence)
- Real-time Notifications (Socket.IO)

## Tech Stack
- Node.js + Express
- MongoDB
- Ethers.js (Blockchain)
- Google Gemini AI
- Socket.IO
